Регистрация через Keycloak

Для создания нового аккаунта необходимо зарегистрироваться в системе Keycloak. Регистрация пользователей осуществляется администратором через консоль Keycloak.

Тестовые учетные записи:
• Логин: user, Пароль: password
• Логин: admin, Пароль: admin

Вернуться ко входу Открыть Keycloak Admin Console

После регистрации в Keycloak вы сможете войти в систему через кнопку "Войти через Keycloak"
